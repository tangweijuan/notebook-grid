# Notebook Grid - 在线网格纸生成器

Notebook Grid 是一个免费的在线网格纸生成工具，帮助你快速创建和下载各种规格的网格纸。

## 功能特点

- **多种网格类型**：支持方格纸、点阵纸、横线纸和竖线纸
- **多种纸张尺寸**：A4、A5、A6、A7 可选
- **自定义间距**：可调整网格间距（1-20mm）
- **自定义颜色**：支持自定义网格颜色和纸张颜色
- **导出格式**：支持导出 PNG 和 SVG 格式
- **实时预览**：即时查看修改效果

## 在线使用

访问 https://tangweijuan.github.io/notebook-grid/ 即可直接使用。

## 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

然后在浏览器打开 http://localhost:3000/notebook-grid/

## 技术栈

- React + TypeScript
- Vite
- Tailwind CSS
